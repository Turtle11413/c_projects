# Pong

## install
```bash
git clone git@github.com:Turtle11413/Pong.git
cd Pong
```

## build
- pong
```bash
make pong
make run_pong
```

- pong interactive
```bash
make pong_interactive
make run_pong_interactive
```

- pong cringe
```bash
make pong_cringe
make run_pong_cringe
```
## controls
 - a - 1st player up
 - z - 1st player down

 - k - 2nd player up
 - n - 2nd player down
